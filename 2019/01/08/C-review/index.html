C#回顾

与java明显不同

  1. WriteLine()
  2. 文件名可以不同于类的名称
  3. 动态（Dynamic）类型
    您可以存储任何类型的值在动态数据类型变量中。这些变量的类型检查是在运行时发生的。 声明动态类型的语法：dynamic <variable_name> = value;
  4. 指针类型变量存储另一种类型的内存地址。C# 中的指针与 C 或 C++ 中的指针有相同的功能。
  5. 常量是使用 const 关键字来定义的
  6. ? : 运算符
    我们已经在前面的章节中讲解了 条件运算符 ? :，可以用来替代 if…else 语句。它的一般形式如下：Exp1 ? Exp2 : Exp3;
  7. for(;;) 结构来表示一个无限循环。
  8. return 语句可用于只从函数中返回一个值。但是，可以使用 输出参数 来从函数中返回两个值。输出参数会把方法输出的数据赋给自己，其他方面与引用参数相似。 关键字 out
  9. 在 C# 中，使用 ref 关键字声明引用参数。
  10. 数组：double[] balance = new double[10];
  11. foreach (int j in n )
  12. C# 参数数组:params 关键字 当声明一个方法时，您不能确定要传递给函数作为参数的参数数目
    在使用数组作为形参时，C# 提供了 params 关键字，使调用数组为形参的方法时，既可以传递数组实参，也可以只传递一组数组。params 的使用格式为：public 返回类型 方法名称( params 类型名称[] 数组名称 )
  13. C# 中的析构函数
    类的 析构函数 是类的一个特殊的成员函数，当类的对象超出范围时执行。
    析构函数的名称是在类的名称前加上一个波浪形（~）作为前缀，它不返回值，也不带任何参数。
    析构函数用于在结束程序（比如关闭文件、释放内存等）之前释放资源。析构函数不能继承或重载。

注意点

  1. C# string 字符串的前面可以加 @（称作”逐字字符串”）将转义字符（\）当作普通字符对待，比如：string str = @"C:\Windows";等价于：string str = "C:\\Windows";
  2. Console.ReadLine() 只接受字符串格式的数据。
  3. 可空类型：在处理数据库和其他包含可能未赋值的元素的数据类型时，将 null 赋值给数值类型或布尔型的功能特别有用：< data_type> ? <variable_name> = null;
  4. 我们可以使用 static 关键字把类成员定义为静态的。当我们声明一个类成员为静态时，意味着无论有多少个类的对象被创建，只会有一个该静态成员的副本。
    关键字 static 意味着类中只有一个该成员的实例。
  5. 您也可以把一个成员函数声明为 static。这样的函数只能访问静态变量。静态函数在对象被创建之前就已经存在。
  6. 抽象类在某种程度上与接口类似，但是，它们大多只是用在当只有少数方法由基类声明由派生类实现时。

待详细看

  1. namespace命名空间:using System;命名空间的设计目的是为了提供一种让一组名称与其他名称分隔开的方式。在一个命名空间中声明的类的名称与另一个命名空间中声明的相同的类的名称不冲突。
  2. 可空类型
  3. 多维数组和交错数组:C# 支持多维数组。多维数组又称为矩形数组。交错数组是数组的数组。
  4. Array 类是 C# 中所有数组的基类，它是在 System 命名空间中定义。Array 类提供了各种用于数组的属性和方法。直接调用array中的方法能实现数组的逆转和排序等
  5. 结构在 C# 中，结构是值类型数据结构。它使得一个单一变量可以存储各种数据类型的相关数据。struct 关键字用于创建结构。
  6. 动态多态性

委托（delegate)

C# 中的委托（Delegate）类似于 C 或 C++ 中函数的指针。委托（Delegate） 是存有对某个方法的引用的一种引用类型变量。引用可在运行时被改变。

委托（Delegate）特别用于实现事件和回调方法。所有的委托（Delegate）都派生自 System.Delegate 类。
