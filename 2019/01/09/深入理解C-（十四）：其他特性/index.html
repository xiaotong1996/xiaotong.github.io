C# 3的分部方法

分部方法的声明与抽象方法相同:

使用partial修饰符无须实现，在实际的实现部分也要partial修饰符。

分部方法可以不被实现，这些未被实现的分部方法的调用会被编译器移除。

由于方法可能不存在，分部方法返回类型必须为void，且不能获取out参数。必须是私有的
