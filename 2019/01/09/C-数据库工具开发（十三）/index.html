C#数据库工具开发（十三）

问题（08/01）

单行同步，容易实现，只要判断选择的行状态做操作就行。
如何撤销？用栈存储已执行的sql指令以及执行前的字段状态，对应每个指定，做反操作。

做同步操作，全过程

  1. 根据差异同步DB
  2. 清空dblists
  3. 取更新后的dblists
  4. 用新的dblists填充dataDB
  5. 更新新的infolists
  6. 按照新的dblists顺序，对infolists排序，并删除已经选择去掉的项
  7. 用新的infolists填充dataxml
  8. 同步索引项
